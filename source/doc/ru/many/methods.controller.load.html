load

(blitz >= 0.4)

load — загрузить тело шаблона из переменной PHP

bool load ( string tpl )

Пример 14. load

<?

$body = <<<BODY
hello, {{ \$who }}!
{{ bye; }}

BODY;

class View extends Blitz {
    function bye() {
        return "Have a lot of fun!...";
    }
}

$T = new View();
$T->load($body);
echo $T->parse(array('who' => 'world'));

?>
hello, world!
Have a lot of fun!...
